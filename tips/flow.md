分類を抜け漏れなく十分考慮できていない場合は (※) を記載

## 質問

1. 「**架空のモノ？**」  
  Yes/Noとは言えないorわからない の場合:
    - 「**現実にあるとしたら○○？**」
    - "架空"の種類: 物語/都市伝説/科学的でない現象,説等/証明されていない現象,説等/... (※)
    - 「**物語の中では○○？**」
    - 「**昔は実在すると信じられていた(ことがある)？**」(一時期信じられていたものも含む)
    - 「**現在では未だに実在が証明されていない？**」
2. 「**原子で構成？**」 (≒物体/概念/情報など？)  
  Yes の場合: \[#モノである場合\] へ  
  No の場合: \[#モノでない場合\] へ  
  にごされた場合:
    - 「**言葉が示す具体的な何かは原子で構成？**」

- 「**単体or一部or集合？**」
  - 一部の場合: 「**全体を示す何かは原子で構成？**」
  - 集合の場合: 「**個々の何かは原子で構成？**」
- ○○に関係ある？ → \[#関係あるモノ探し\] へ  
  ※人によっては関係性が遠いキーワードてもYesになる可能性があるので注意  
- 成立年代:
  - 言葉が表す何かの成立(通常)年代
  - 言葉自体が生み出されたor成立した年代
  - 言葉が示す具体的な何かが初めて成立した年代
  1. 「**WW2終戦後に成立した？**」
  2. 「**文明が興る前からあった？**」
  3. 「**現在でもある？**」
  4. 「**人間がいなくでも成立する？**」
  5. 以降は \[#年代の分類\] へ
- 「**唯一？**」  
  (Yesの場合: 「**固有名詞？**」)

### モノである場合

- 「**肉眼で直接見える？**」  
  Noの場合: 「**顕微鏡等を用いたら見える？**」,「**望遠鏡等？**」, ... (※)
- 「**現在でも生きている？**」  
  Yesの場合:
  - 「**人間, 人間の一部, または人間の集合？**」  
    - 人間の場合: 立場/職業/有名人かどうか/出題側との関係/... (※)
    - 人間の集合の場合: 団体/企業/... (※)
- 「**買える？**」(≒金に替えられる価値を持つ？)
  - Yesの場合:
    - 金額: 「**税抜○○円以内で買える**？」  
      100万/1万/100/... (logスケールで細かく)
    - 場所: 「**○○で買える？**」
      - 目的別: スーパーマーケット/家電量販店/ホームセンター/ドラッグストア/家具店/スポーツ用品店/カー用品店/自転車用品店/服屋/郵便局/おもちゃ屋/装飾屋/... (※)
      - 総合的に商品を取り扱う店: 100均/ショッピングセンター(イオン/イオンモール)/... (※)
      - その他: 不動産屋(建物は別方面から)
      - 「**専門店がある？**」  
        Yesの場合: 眼鏡orコンタクト/靴/... (※)
  - "頑張れば買えるかもしれないが普通はNo" の場合:  
    (金額は？: 100億/1兆/10兆/100兆/... → 採用するか微妙,これがわかっても絞りにくいので)
- 「**大きさが定義できる？**」  
  Yesの場合:
  - サイズ: 指で持てる？/片手では？/両手では？/人間/建物/都道府県/国/地球  
    小さい場合:
    - 「**身につけるモノ？**」
    - 「**あなたor私は今持っている？**」
- 場所: 「**屋内にある？**」  
  - Yesの場合: 場所, ... (※)
    - 「**家or実家にある？**」  
      Yesの場合:
      - 部屋: 自室/台所/洗面所/玄関/トイレ/風呂/リビング/ダイニング/...
    - 「**小学校or中学校or高校にある？**」  
      Yesの場合: 教室/廊下/体育館/運動場/... (※)
  - Noの場合:
    - 場所: 道路/駐車場/歩道/公園/公共的な敷地(駅/空港/港/...)/山/海/空/個人の敷地(庭/...)/... (※)
  - 屋内にも屋外にもある場合: 「**身につけるor持ち歩く？**」
- 「**手で直接触れる？**」
- 三態: (参考: [物質の状態 - Wikipedia](https://ja.wikipedia.org/wiki/%E7%89%A9%E8%B3%AA%E3%81%AE%E7%8A%B6%E6%85%8B), [相 (物質) - Wikipedia](https://ja.wikipedia.org/wiki/%E7%9B%B8_(%E7%89%A9%E8%B3%AA)))  
  - 「**(室温で)固体っぽく見える？**」(外側部分or主要素が)
    Yesの場合: 固体/[アモルファス](https://ja.wikipedia.org/wiki/%E3%82%A2%E3%83%A2%E3%83%AB%E3%83%95%E3%82%A1%E3%82%B9)([ガラス](https://ja.wikipedia.org/wiki/%E3%82%AC%E3%83%A9%E3%82%B9)等)/粘度の高い液体([アスファルト](https://ja.wikipedia.org/wiki/%E3%82%A2%E3%82%B9%E3%83%95%E3%82%A1%E3%83%AB%E3%83%88)等)/[ゲル](https://ja.wikipedia.org/wiki/%E3%82%B2%E3%83%AB)(こんにゃく/ゼリー/豆腐/シリカゲル/etc.)/  
    ※氷の大量の相はだいたい固体
  - 「**(室温で)液体っぽく見える？**」  
    Yesの場合: 液体/[ゾル](https://ja.wikipedia.org/wiki/%E3%82%BE%E3%83%AB)(牛乳/マヨネーズ/石鹸水/etc.)
  - 「**(室温で)気体っぽく見える？**」  
    Yesの場合: 気体/[エアロゾル](https://ja.wikipedia.org/wiki/%E3%82%A8%E3%82%A2%E3%83%AD%E3%82%BE%E3%83%AB)(煙/霧orもや/スモッグ(PM2.5等)/etc.)  
  - その他の物質の相(っぽいもの): [プラズマ](https://ja.wikipedia.org/wiki/%E3%83%97%E3%83%A9%E3%82%BA%E3%83%9E)/[ボース=アインシュタイン凝縮](https://ja.wikipedia.org/wiki/%E3%83%9C%E3%83%BC%E3%82%B9%EF%BC%9D%E3%82%A2%E3%82%A4%E3%83%B3%E3%82%B7%E3%83%A5%E3%82%BF%E3%82%A4%E3%83%B3%E5%87%9D%E7%B8%AE)/リュードベリ状態/String-net liquid/[フェルミ縮退](https://ja.wikipedia.org/wiki/%E3%83%95%E3%82%A7%E3%83%AB%E3%83%9F%E7%B8%AE%E9%80%80)/量子ホール状態/...  
  - 怪しいもの: [超流動](https://ja.wikipedia.org/wiki/%E8%B6%85%E6%B5%81%E5%8B%95)状態/[超固体](https://ja.wikipedia.org/wiki/%E8%B6%85%E5%9B%BA%E4%BD%93)/液晶→どっちととる?/超ガラス??/...  
- 「**材質はこれと言える？**」  
  Yesの場合: 木(植物)/金属/プラスチック/紙/石/布/ガラス/... (※)
- 「**道具？**」(≒使う？)  
  Yesの場合: 場所/時間帯/用途/... (※)
  - 「**家事で使う？**」
  - 「**工具？**」
- 「**飲食可能？**」  
  Yesの場合: 料理/飲料/食材/調味料/... (※)
- 「**色は決まっている？**」

### モノでない場合

\[#モノである場合\] も質問も使える

- 「**コンピューターorインターネット上にある？**」  
  Yesの場合:
  - 「**コンピュータ上で見える？**」
  - 「***コンピュータ上で実体を持つ？***」
- 「**大きさを定義できる？**」
- 「**五感で感じることができる？**」  
  - Yesの場合:
    - 「**見える？**」  
      Yesの場合: 現象/行為or行動/サービス/団体/スポーツ/映像/地域or地名/イラスト(キャラクター等)or図(ロゴ等)/... (※)  
      - 媒体: テレビ/スマートフォン/インターネット/舞台/... (※)
      - 「**文字情報を含む？**」
    - 「**聞ける？**」  
      Yesの場合: 音(音楽/ラジオ/警報/...) (※)
    - 「**五感のどれと言える？**」  
      Noの場合: "季節を感じる" のような言い回しも考慮する
  - Noの場合: (※)
    - 存在性
      - 定常的な何か: /状態/性質or特徴/...
      - 条件が変わらなければ定常的
      - 変化を含む何か: 現象/...
      - 動作を含む何か: 行為or行動/スポーツ/...
    - 時間の流れ?
      - (ある過去から現在まで)継続的: 地域or地名/...
      - ある一点or範囲の時間が基準: 出来事orイベント/時間自体/...
      - 定期的/一回きり(→唯一)/不定期: ...
    - 人間の内面: 感情/性格/思考/...
    - 概念: 分類or分類の1つ/関係/...
      - 分類の場合: 分類の対象
    - その他: 団体(企業/音楽グループ/宗教/...)/サービス/権利/地位or立場/自然...
    - ゲームタイトル,方法or手段,表現

### 関係あるモノ探し

- 「**あなたor一般人と関係ある？**」  
  Yesの場合: 時期, 頻度
  - 「**あなたは○○以内に関わった？**」: \[#年代の分類\] へ
  - 「**○に○回以上関わる？**」  
    週に4回/月に1回/年に1回/...
- 地域
  1. 「**日本特有？**」  
  2. 「**世界中である？**」
  3. 「**特定の国と関係ある？**」  
    Yesの場合: \[#地域の分類\] へ
  4. 「**特定の文化圏と関係ある？**」
- 「**(特定の)人間と関係ある？**」 (※人によってはほぼすべてYes)  
  Yesの場合: 関係ある人間の種類, 人数など  
  - 「**特定の職業と関係ある？**」  
    Yesの場合: サービス業/一次産業/二次産業/... (※詰め切っていない)
  - 「**特定の年齢層と関係ある？**」  
    Yesの場合: 子ども(小中高)/大学生/大人/... (※)
  - 「**一人でも成り立つ？**」
  - 「**複数人で成り立つ？**」
    Yesの場合:
    - 「**二人で成り立つ？**」  
      Yesの場合: 「**二人の関係性が重要？**」, ...
  Noの場合:
  - 「**人間が存在しなくても成り立つ？**」
- 日常
  - 「**仕事or生活と関係ある？**」  
    Yesの場合: (※)
- 「**娯楽と関係ある？**」  
  Yesの場合: 娯楽の分類 (場所, 行為タイプ) など  
  (参考: [Category:娯楽 - Wikipedia](https://ja.wikipedia.org/wiki/Category:%E5%A8%AF%E6%A5%BD))
  - 場所
    -「その娯楽は**屋内？**」  
      Yesの場合: 音楽/映像(ドラマ/アニメ/映画/YouTube/...)/舞台/ゲーム/本(小説/漫画/...)/各種クリエイター/SNS/料理or食事/ギャンブル/... (※)
    - 「その娯楽は**屋外？**」(≒アウトドアに近い?)  
      Yesの場合: アウトドアスポーツ/キャンプ/釣り/ドライブ/自転車/旅行/ガーデニング/DIY/... (※)
  - 行為/タイプ
    - 「その娯楽は**運動を含む？**」  
      Yesの場合: スポーツ/登山/マリンスポーツ系/... (※)
    - 「その娯楽は**運動と関係ある？**」  
      Yesの場合: スポーツ/スポーツ観戦/... (※)
    - 「その娯楽は**クリエイティブなことをする？**」(※)
- 「**学校or学問と関係ある？**」  
  Yesの場合: "学校or学問"が広いのでさらに絞り込み
  - 「**学問と関係ある？**」  
    Yesの場合: [学問の分類ページ](./academic-discipline.html) を参照
    - 「**習った？**」  
    Yesの場合: 「**○○までに習った？**」→小学校卒業/中学校/高校or高専/専門or短大or大学/大学院
    - 「**複数の学問と関係ある？**」
  - 「**○○(具体的な学校)と関係ある？**」  
    高校まで/予備校/大学/自動車学校/特定の専門学校(調理/理容or美容/看護/保育/福祉/デザイン(服飾含む)/音楽/商業/工業/農業/...)/...  
    参考: [専修学校の8分野について -一般財団法人 職業教育・キャリア教育財団](https://www.sgec.or.jp/scz/foundation/system/8channel.html)
  - 「**学校生活と関係ある？**」
  - 「**学校行事と関係ある？**」
  - その他 (※)
- 「**インターネットorコンピューターと関係ある？**」  
  Yesの場合:
  - 「**インターネットで見える？**」  
    Yesの場合:
    - 「**SNSで見える？**」
  - その他 (※)
- 「**宗教と関係ある？**」  
  Yesの場合: (※)
- 「**法律と関係ある？**」  
  Yesの場合: (※)
- 「**公益的な機関orサービスと関係ある？**」  
  Yesの場合: 役所/鉄道/郵便/放送/... (※)
- 「**時間帯or時期(季節)と関係ある？**」
  Yesの場合: (※)

## その他の質問

- 「**多義語？**」  
  Yesの場合: 何らかの分類にもなり、1つの単体を表すこともあるかもしれない  
  例: 政治家(職業の一分類/特定の人間:○○さんは政治家です), くじ引き(行為/引いたくじ自体)  
  - 「**モノとも概念とも捉えられる？**」  
    Yesの場合: *モノとして考えた場合と概念として考えた場合を分けて質問する*
- 「**学校で習う？**」 (採用するか微妙)  
  Yesの場合: \[#関係あるモノ探し\]→学校→[学問](./academic-discipline.html)) と似ている
- 「**Wikipediaに記事がある？**」 (その人に親しい人間などの判別には役立つが汎用的でない?)
- 「**情報量を持つ？**」
