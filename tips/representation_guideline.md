## 表現について

### 英数字, 記号

- アルファベット: 正式名称が全角を用いる場合を除いて半角アルファベットのみ許容
- 数字:
  - 金額, 各単位系の数値: 半角アラビア数字 `(?<!WW)([\.,]?(\d+))+(?![\.円年週月日時分問回人ヶ]|文字)(?=[亜-熙ぁ-んァ-ヶ])`
  - 「一つ目」,「二つ目」など: 漢数字
- 「！」or「!」: 基本的に「!」のみ許容
- 「？」,「?」,「??」or「???」:
  後ろにスペースは基本含まない `/?\s/`
  - 質問末尾: 「？」 (※これ以外には使わない) `/？([^→]|$)/`
  - 軽い疑問: 「?」
    - 確定させるor確定できる情報かどうか怪しい場合
  - 疑念: 「??」
    - 人によって解釈が異なることが懸念される場合
  - 強い疑念: 「???」
    - 不確実、信憑性に欠ける情報の場合
  - ノリ: ?の数は任意
- 「()」(半角括弧) or「（）」(全角括弧): 基本的に「()」(半角括弧) のみ許容?? (※)
- 「、」,「,」or「/」: `\d+\.\s.*、.*？→`
  - 文章中の読点: 「、」
  - Yes or No と文章の区切り: 「,」(半角スペースを含めない)
  - 単純列挙: 「, 」(半角スペースを含める) (例: 一つ目, 二つ目, 三つ目)  
    ただし直後が全角の括弧類の場合
  - フロー中の列挙: 「/」
  - README中の列挙: 「、」
  - 質問文中でいずれかを問う際の列挙: 「,」(日本語のみ半角スペースを含めない?)
    - 日本語の場合: 一つ目,二つ目,三つ目or四つ目
    - 英語の場合: A, B, C or D
- ダブルクォーテーション(""),カギ括弧(「」) or 白抜きカギ括弧(『』):
  - 答えの表記: 『』
  - Wikipediaの記事名, カテゴリ名: 「」
  - 表現: 「」
  - 引用: 「」
  - (しばしば直前の)強調, 用法の注意: ""
- 「×」or「x」: 基本的に「x」のみ許容, 半角英字との間のみ半角スペースを挿入(半角数字との間は挿入しない) `[^a-zA-Z]x[^a-zA-Z]`
- 「...」(ピリオドx3) or「…」,「……」:
  - 列挙の末尾: 「...」
  - 軽い言いよどみ: 「…」(※)
  - 言いよどみ: 「……」(※)
- 「・」,「&」,「＆」or「and」: 使わない?? (※未定) `(?<!\()・`
- 「～」: 省略
- 「※」: 検討事項がある場合のみ付随を許容
- 「→」: 質問と回答の区切り, 詳細の分類 (※)  `(?<!？|ギブアップ)→`

### 表記揺れ

- 「モノ」,「もの」or「物」: (※「物」は基本的に非許容?)
  - "存在"を指す場合: 「モノ」(例: )
  - その他: 「もの」
- 「or」,「または」,「もしくは」or「や」:  
  - 普通の場合: 「or」のみ許容
  - ネストする場合: 一段階目に「や」, 二段階目に「または」or「もしくは」, 一番外側は「or」のみ許容??
- 「など」,「等」or「etc.」:
  - 質問中: なるべく「等」を使いたい
  - フロー中: ※未定
- 「あなた」,「私」,「我々」or「私たち」: `(私|あなた)(?!\(|たち)`
  - 質問時: 「あなた(出題側)」「私(推理側)」と表記
  - 回答時: 「あなた(推理側)」「私(出題側)」と表記
- 答えを指す「それ」or「お題」: その表現を消す or「答え」のみ許容
- 「1つ」or「一つ」: 基本は「一つ」のみ許容?? (※) `\dつ`
- 「...」,「など」or「etc.」: ※未定
- 「手のひら」: 「**手のひらサイズ？**」と問う場合以外は基本的にはっきりさせて「両手のひら」or「片手のひら」と表記
- アルファベットと日本語の単語間: 半角スペースを挟む
  - (※ (○○) or ～)
- 例示: 「(例: ...)」と表記???
- 「関係ある」,「関係する」,「関係がある」,「関連ある」or「関連する」: 「関係ある」のみ許容 `関[係連]`  
  (※「関連づける」,「関連項目」,「関連リンク」,「関連性」,「関連」などはそのまま?)
- 「する」or「行う」: 積極的に「行う」への読み替えを行う (※「行う」のみ許容?)
- 「ある」or「存在する」: 積極的に「存在する」への読み替えを行う?? (※「存在する」のみ許容?) `(?<!関係)ある？`
- 「ある」or「ありうる」: 「あり得る」へ積極的に読み替える
- 「○○で買える？」,「○○で売っている？」or「○○に売っている？」: 「○○で買える？」のみ許容 `[にで]売っている`
- 「スーパー」or「スーパーマーケット」: 「スーパーマーケット」のみ許容 `スーパー(?!マーケット)`
- 「言う」or「呼ぶ」: "呼びかける" の意で使う場合を除き「言う」のみ許容
- 「たぶん」or「おそらく」: どちらも許容
- 「可能」or「できる」:
  - 質問or回答中: 「可能」のみ許容
  - フロー中: 「できる」のみ許容
- 「○○(動詞)ことができる」or 「○○れる/○○られる」: 「～ことができる」のみ許容
- 「○○することができる」or「○○可能」: 「○○可能」のみ許容
- 「とき」or「場合」: ※未定
- 「素材」,「材質」or「材料」: 言葉が意味するところは若干異なるが、なるべく「素材」を使いたい
- 「持つ」or「所有する」: なるべく「所有する」を使いたい
- 「主な」or「主要な」: 「主な」のみ許容
- 「問う」,「尋ねる」or「訊く」: 「問う」のみ許容
- 「連想(する,できる,可能)」or「イメージ～」: 「連想～」のみ許容?? `イメージ(?!では|する|[はをの]|$)`
- 「けど」or「が」: どちらも許容
- 「=」,「＝」,「≒」or「すなわち」: 基本的に前後にスペースは挟まない (「＝」「すなわち」は非許容??)
  - 同義, 意味の言い換え: 「=」
  - ほぼ同義, 文意の言い換え: 「≒」

#### ひらがなのみ許容する表現

表外読みは原則ひらがなにする (※公文書の書き方にある程度従う??)

| 許容 |非許容 | (備考) |
|:-:|:-:|:-:|
| あるいは | 或いは |
| (～とorで) いう | 言う |
| おそらく | 恐らく |
| および | 及び |
| こと | 事 | 形式名詞 |
| たぐい | 類い, 類 | "るい" の読みと見分けがつかないため |
| ただし | 但し |
| ～たち | ～達 | 接尾語 (例: 私たち) |
| たぶん | 多分 |
| できる | 出来る |
| とき | 時 | 形式名詞 |
| ときどき | 時々 |
| など | 等 | "とう" と読まれる, これは表外読み |
| ほう | 方 | 形式名詞, 比較の場合 |
| ほか | 他, 外 |
| まだ | 未だ | "いまだ" と読む場合は漢字 |
| または | 又は |
| もしくは | 若しくは |
| もの | 物, 者 | 形式名詞, 表記揺れの項も参照 |
| (～して)よい | 良い | 補助用言, 接尾語 |
|  |  |

- とき:  `(?<![一当]|開始|終了|出現|成立|移動|緊急)時(?![期間代計点差])`
- ほか:  `(?<!その)他(?!人)`

#### 漢字のみ許容する表現

| 許容 |非許容 | (備考) |
|:-:|:-:|:-:|
| あり得る | ありうる | "ありえる" はもともと誤用 |
| ～し得る | ～しうる |
| 全て | すべて |
| 何か | なにか |
| 私 | わたし | もともとは慣用表現だが常用漢字表の読みに追加された |

#### そのほかに間違えそうな言い換え

- ことある→ことがある `ことが(?!あ(る|りそう)|ない|可能|多い)`

#### 検索用正規表現

- `言(?![葉語]|える|い方)`
- `(?<![購])入(?![学])`
- `(?<!ことが|関係)ある？`

### 質問フォーマット

- 頭に通し番号を「1. 」,「2. 」, ... のように付記する
- 通常の質問:
  - 「(質問)？→(答え)」と表記 `\d+\\?\.\s*[^？]*？→`
  - 敬語は利用しない
  - 答えの表現はある程度口語的でよい
- 答えを確認する質問: (\d+\.)(.*)(?<!\s答え[はが]|\s)(『.*』.*？)`
  - 1回目:「答えは『(答えの単語)』？→(回答)」と表記  
    ただし直前の質問で出た単語やそれに非常に近い単語を答えとして問う場合は「答えが『(答えの単語)』？」と表記してもよい
  - 2回目以降: 上記のほかに「『(答えの単語)』？→(回答)」と表記してもよい
- 過去の質問番号の参照: 「1.」とする (例: 「12.の回答理由は...」)
- Wikipedia を参照する場合:
  - 記事自体: 「[(記事名)](link)」
  - 記事カテゴリ: 「[(カテゴリ名)](link)」(例: 記事カテゴリに「[(カテゴリ名)](link)」を含む？)
- "○○する" という動詞が最後につく質問: 「○○する？」と直接問う場合を除き "する" を省略して「○○？」と表記
  - (例1: 「昔から存在する？」→「昔から存在？」)
  - (例2: 「屋内で利用する？」→「屋内で利用？」)
  - (例3: 「移動する？」→そのまま)
- 同じような質問を連続で問う場合: 直前と同じなら文末を省略してもよい
  - (例1: 「ゲームに登場？」→「漫画には？」→「アニメには？」)
  - (例2:「地学or生物と関係ある？」→「化学？」→「物理？」)
- 出題側へイメージ・第一印象を問う場合: (例: 「○○しそう？」) といった表現を使う
- 出題側からの誘導やミス?により質問以外から得られる情報が含まれる場合は質問の頭に「(ヒント) 」を付記, 質問→回答の体をなしていなくてもよい
- 答えを既に知っている人間からの質問は頭に「(インサイダー) 」を付記 (※インサイダーゲームから引用してきた名称) `\d+\\?\.\s*\((?!オーディエンス|インサイダー|ヒント).*\)`
- 答えの言い直し: 「～？→(当初の答え)…やっぱり～」と表記することが多い `(?<!…)やっぱり`
- ギブアップした場合は、箇条書きとは切り離して「ギブアップ→答えは『○○』」と表記

### 具体的な質問に関するフォーマット

- 架空の存在かどうかを問う質問: 「**架空のモノ？**」 `架空(?!のモノ|上)`
- 何で構成されているかを問う質問: 「**○○で構成？**」
- 架空上における質問: 「**架空上の～？**」,「**架空上で(は)～？**」()
- 素材,材質or材料を問う質問:
  - 「**(主な)素材を定義可能？**」
- 関係あるかどうかを問う質問: `に(?=関係)`
  - 質問中: 「**～と関係ある？**」(※「**～に関係？**」に変更??)
  - フロー中: 「**～と関係ある？**」
- 大きさを問う質問:
  - 「**大きさを定義可能？**」 `大きさ[がは](定義可能|ある)` / `大きさ(?!を定義可能)`
  - 「**手のひらサイズ？**」

## 配信時について

- 質問と回答が画面内の1行に収まらない場合:
  - まずは質問の末尾(？→の直後)に改行を挟む `/？→$/`
  - 改行後はインデントを揃える
  - それでも収まらない場合は適宜
  - ログへの記載時は改行とインデントを消す
- 答えがおしい場合: 特記事項がない場合は以下の通りに表記
  - 1回目:「オシイ!」or「オシイ!No」
  - 2回目以降:「オシイ!」,「オシイ!No」のほかに「オシイ,No」と表記してもよい
- 配信コメントからの質問は頭に「(オーディエンス) 」を付記
